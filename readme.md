# Censor Bot

Telegram бот для автоматической модерации чатов с использованием нейросетевой модели для определения токсичности сообщений.

## О проекте

Бот анализирует сообщения в группах, выявляет токсичный контент и принимает меры:
- Удаление явно токсичных сообщений
- Отправка подозрительных сообщений на проверку администраторам
- Выдача предупреждений нарушителям
- Автоматическая блокировка после 5 предупреждений

## Возможности

- **Автоматическая модерация** — анализ сообщений с помощью обученной нейросети
- **Система предупреждений** — накопление предупреждений с автоматическим баном
- **Администрирование** — уведомления администраторам с кнопками для принятия решений
- **Персистентность** — все данные сохраняются в SQLite базе данных
- **Восстановление статуса** — автоматическая синхронизация прав бота при перезапуске

## Технологии

- **Python 3.13.5**
- **aiogram** — асинхронный фреймворк для Telegram Bot API
- **PyTorch** — нейросетевая модель (CNN + BiLSTM)
- **SQLAlchemy** — ORM для работы с базой данных
- **SQLite / PostgreSQL** — хранение данных

## Запуск

- Добавьте в файл **.env** **BOT_TOREN** - ключ Telegram бота и **DB_URL** - путь к вашей базе данных
- Откройте терминал в папке, в которой лежит проект и выполните команду: **python bot.py**. Боти начнет свою работу в тот же момент
